About us

This application was realized by Valerio Cappuccio as an experiment and demonstration of the integrating of a system dynamics model into a web page. It is not an official page and is still work in progress. The model is abstract and not initialized with real data, so any output is dummy data.

The system dynamics model used is the publicly available Community Coronavirus Conceptual Model Tom Fiddaman, Ventana Systems, Inc., 2020.

The application uses a Flask server to render .html templates and run in the background the Python version of the model.

The system dynamics model was converted into Python using PySD library developed by James P. Houghton.
